Covid 19 Future Predictor Application for India

This is a web application designed in order to predict the state of our country which is currently going through a pandemic situation. The main goal of the project is give state wise predictions about the rise/fall of cases of the virus along with attempting to predict the 'Deadline for the 2nd wave' in India.
